ℹ️ Cara Mengaktifkan Payment Gateway
1. Isi API Key / credential dari dashboard masing-masing provider.
2. Klik toggle ON untuk mengaktifkan provider tersebut.
3. Klik Simpan pada setiap provider.
4. Klik Test untuk memverifikasi koneksi.
5. Update file .env dengan credential yang sama lalu restart server.
